# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by opening a GitHub security advisory for this repository:

https://github.com/lucasrosaalves/industrial-model-ts/security/advisories/new

Include the affected version, reproduction steps, impact, and any known workaround. Avoid public issues for undisclosed vulnerabilities.

## Supported Versions

Security fixes are provided for the latest published version.
